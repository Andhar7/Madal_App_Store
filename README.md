# 🎵 Madal - Sri Chinmoy Spiritual App

![Platform](https://img.shields.io/badge/platform-iOS-lightgrey)
![Swift](https://img.shields.io/badge/Swift-5-orange)
![SwiftUI](https://img.shields.io/badge/SwiftUI-blue)
![Status](https://img.shields.io/badge/status-In%20App%20Store%20Review-yellow)

A comprehensive iOS app providing access to Sri Chinmoy's teachings, music, books, videos, and global peace initiatives.

## 📱 Features

- **Main** - Sri Chinmoy official website
- **Library** - 21,300+ books and writings
- **Races** - Sri Chinmoy Marathon Team events
- **Centre** - Meditation centers worldwide
- **Radio** - Sri Chinmoy Radio streaming
- **Videos** - 700+ spiritual videos
- **Songs** - 21,300 songs composed by Sri Chinmoy
- **Peace Run** - Global peace torch relay
- **Favourites** - Save favorite pages for quick access

## 🛠️ Technical Details

- **Platform**: iOS 15.0+ / iPadOS 15.0+
- **Language**: Swift 5
- **Framework**: SwiftUI
- **Architecture**: MVVM
- **Persistence**: Core Data
- **Web Integration**: WebKit (WKWebView)
- **Design**: Custom UI with side menu navigation
- **Universal**: Optimized for iPhone and iPad

## 🚀 Installation

### Requirements
- Xcode 13.0 or later
- iOS 15.0 or later
- macOS 12.0 or later (for development)

### Steps
1. Clone this repository
2. Open Madal.xcodeproj in Xcode
3. Build and run on simulator or device
